This is the type specimen poster assignment that I did for ATLS 2300: Type. The poster is for the font "News Gothic". I used a newspaper format to illustrate the use of the News Gothic font in print newspaper. Some other examples of things that use News Gothic font are the band ABBA, Lady Gaga, Alfred Hitchcock, and Star Wars to name a few.

This is an animation assignment that I did for ATLS 2100: Image. The assignment was an animated haiku, which we could generate ourselves or use something else. I realised that "I'm a Believer" by: Smash Mouth, which was featured in Shrek, had the same rhythm pattern that was needed in a haiku, and decided to animate the haiku with the soundtrack in the back.
